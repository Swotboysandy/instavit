# Building the AI Screenshot Assistant

## For Windows Installation

### Option 1: Portable Executable (Recommended - No Installation Required)
1. Open terminal in the app directory
2. Run: `npm run build:portable`
3. Find the portable .exe file in the `dist` folder
4. **Copy this .exe to any computer** and run it - no installation needed!

### Option 2: Full Installer (With Desktop Shortcut)
1. Open terminal in the app directory
2. Run: `npm run build:win`
3. Find the installer in the `dist` folder
4. Run the installer on any Windows computer

## Quick Start (Development Mode)
```bash
npm install
npm start
```

## Features
- ✅ **Draggable icon** - Move it anywhere on your screen
- ✅ **Auto-analyze** - Automatically describes screenshots
- ✅ **MCQ support** - Identifies correct answers with explanations
- ✅ **Black & white UI** - Professional monochrome design
- ✅ **Click outside to minimize** - Easy to hide
- ✅ **Always on top** - Works over any application

## System Requirements
- Windows 10 or later
- No additional software needed for portable version
- Internet connection for AI analysis

## Distribution
The portable .exe can be:
- Copied to USB drives
- Shared via email or cloud storage
- Run on any Windows PC without installation
- No admin rights required
